How this version differs
- Nigeria is intentionally excluded from 07_administration_level_1.csv because you already built Nigeria more extensively in a separate Nigerian ancestry dataset.
- The other 27 countries from the diaspora-origin country list are retained.
- This is an ancestry-focused Level 1 file, not a full legal/administrative gazetteer for every possible place in the world.

Selection method
1) For countries where indigenous/rooted populations are distributed widely across the national territory,
   all official current first-level units were retained.
2) For larger modern states with many administrative units (for example the USA, Canada, Brazil, Spain, Portugal),
   only selected Level 1 units were retained where later Level 3 mapping is more likely to support:
   - ethnic/cultural identity
   - migration lineage
   - local community mapping
   - Atlantic slave trade cultural diaspora
   - trade migration

Important modelling note
- Administration Level 1 is only the first ancestry filter.
- The true specificity of origin will come at Administration Level 2 and especially Administration Level 3
  (district/county/parish/town/city/settlement).
- This means some countries are intentionally broad at Level 1 so you can narrow correctly at Level 2/3.

Research caution
- This file is a working genealogy dataset, not a political or legal statement about identity.
- For countries marked as selective in the country summary, treat this as a curated v1 origin map to refine later.
