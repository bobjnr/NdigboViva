ADM3 model used:
- ADM1 = Province / Area
- ADM2 = District
- ADM3 = Chiefdom

Batch result:
- 190 ADM3 rows added for Sierra Leone
- Western Area Rural District and Western Area Urban District intentionally receive no ADM3 rows because Sierra Leone's chiefdom layer applies to the rural districts, while the Western Area is administered outside the chiefdom structure.

Important note:
- Public sources agree that Sierra Leone has 16 districts and 190 chiefdoms, but spellings vary across sources.
- The Sierra Leone Web current list was used as the operational name source for district-to-chiefdom mapping.
- One Karene entry was normalized to "Mafonda Makerembay" because the source list breaks that name across lines.
