IGBO DIASPORA ORIGIN - ADMIN LEVEL 3 BATCH 25 (GUYANA)

Country:
- Guyana

Model used:
- ADM1 = Region
- ADM2 = Neighbourhood Democratic Council / municipality seed
- ADM3 = anchor settlement seed

What this batch does:
- Adds one ADM3 anchor beneath each existing Guyana ADM2 row already present in 08_administration_level_2.csv.
- Uses the primary place-name embedded in each NDC or municipality label as the ADM3 settlement anchor.
- Keeps the file ancestry-ready without pretending to be a complete village gazetteer.

Important scope note:
- This is a seed_partial batch.
- Potaro-Siparuni and Upper Takutu-Upper Essequibo remain incomplete at ADM3 because the current project file does not yet contain a stable ADM2 parent layer for those paths.
