Batch 7 - Liberia

Model used
- ADM1 = County
- ADM2 = District
- ADM3 = City/Clan/Township seed

Important note
This batch is intentionally marked seed_partial. Liberia's formal third level is commonly described as clans beneath districts, with Greater Monrovia using community/ward patterns instead.
For this production seed, ADM3 rows were generated from the NEC 2023 Phase One VRC file by taking unique City/Clan/Township labels and linking them only where the district name matched an existing ADM2 row in the project dataset.
Covered counties in this source: Bomi, Gbarpolu, Grand Bassa, Grand Cape Mount, Margibi, Montserrado.
