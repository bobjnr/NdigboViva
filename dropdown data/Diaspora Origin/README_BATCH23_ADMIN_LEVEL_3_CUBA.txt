BATCH 23 - CUBA - ADMINISTRATION LEVEL 3

Country: Cuba (CT_CUB)

Hierarchy used
- ADM1 = Province / Special Municipality
- ADM2 = Municipality
- ADM3 = Municipality-equivalent anchor seed

Build logic
- For each existing Cuba ADM2 municipality row, one ADM3 anchor row was created using the municipality name as the ancestry-ready place anchor.
- Isla de la Juventud is handled separately because it is a special municipality at provincial level. It therefore receives a direct ADM1->ADM3 anchor row: Nueva Gerona.

Important note
- This batch is a seed layer for genealogy dropdown use. It is not a full consejo popular, barrio, village, or neighborhood gazetteer for Cuba.
